## Core Features

### 1. User System

- Registration/Login
- Profile Management
- Avatar Upload

### 2. Game System

- Game Lobby
- Game Room Creation
- Real-time Game Battles

### 3. Social Features

- Real-time Chat
- In-game Communication

### 4. Data Storage

- PostgreSQL Database
- User Data Persistence
- Game State Management

## Tech Stack

- Frontend: TypeScript, WebSocket
- Backend: Node.js, Express
- Database: PostgreSQL
- Template Engine: EJS
- Styling: CSS Modules

## Development Tools

- Husky (Git Hooks)
- ESBuild (Build Tool)
- Node-PG-Migrate (Database Migration)
